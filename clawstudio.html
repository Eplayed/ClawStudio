沙盒功能依赖 Docker 来创建隔离的桌面环境。请按以下步骤安装。

  1. Windows: 下载并安装 Docker Desktop for Windows，安装时勾选 WSL 2 后端。安装完成后重启电脑。
  2. macOS: 下载并安装 Docker Desktop for Mac（Apple Silicon 版或 Intel 版），拖入 Applications 后启动。

  4. 安装完成后，确认 Docker 运行中：在终端执行 docker --version，应返回版本号。
